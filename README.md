### gengEditor 的 electron 版本

##### 功能

1. geng-editor 的所有功能
2. 查看使用文档
3. 查看输出 html 字符串
4. 功能快捷键
5. 复制文本功能

##### 技术栈：`electron、@electron/remote、electron-store、electron-builder、geng-editor、react`

##### 运行：

1. 开发启动：npm run dev
2. 命令行科学上网：export http_proxy="http://127.0.0.1:7890"
3. 打包：npm run dist
